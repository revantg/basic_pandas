Nomenclature

Each dataset in this database has a Quandl code of the following formats:

WWDI/{COUNTRY}_{INDICATOR}

where WWDI represents the Quandl database code, {COUNTRY} is the ISO 3-letter code of the country in question, and {INDICATOR} is the indicator code.

Here are some examples showing how the dataset codes are constructed:

IndicatorCountryFull Quandl Code
Access to electricity (% of population)WorldWWDI/WLD_EG_ELC_ACCS_ZS
Adjusted net national income (annual % growth)WorldWWDI/WLD_NY_ADJ_NNTY_KD_ZG
